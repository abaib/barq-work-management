User Roles & Permissions - Part 1

Eight specialized roles provide comprehensive organizational coverage with least-privilege access principles, ensuring users have exactly the permissions needed for their responsibilities while maintaining security and operational efficiency.

Admin

Key Responsibilities
  • System configuration and user management
  • Global settings and security policies
  • Full access to all modules and data
Primary Modules
  • All modules with unrestricted access

Project Manager

Key Responsibilities
  • Project oversight and resource allocation
  • Work order management and scheduling
  • Team performance monitoring
Primary Modules
  • Work Orders, Materials, Reporting, Employee Management

Field Technician

Key Responsibilities
  • Task execution and work completion
  • Mobile field operations with offline capability
  • Photo documentation with GPS tagging
Primary Modules
  • Field Operations, Work Orders, Material Requisitions

QC Inspector

Key Responsibilities
  • Quality verification and inspection checklists
  • Approval or rejection of completed work
  • Mandatory photo evidence collection
Primary Modules
  • QC & HSE Compliance, Work Orders, Field Operations
